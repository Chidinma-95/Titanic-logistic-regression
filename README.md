# Titanic-logistic-regression
My first ML project using Titanic dataset.
Project Overview: This project built a machine learning model to predict passengers survival on Titanc using structurd data. This is my fist end to end data science project using classification technique. The goal is to classify if a passenger survived(1) or did not survve(0) based on features like age, sex, ticket class and fare.
Dataset: The dataset is gotten from Kaggle Titanic competition. Train data contains survival label and the test data used for final prediction.
Encoded variable: Sex-0/1, Embarked-numerical values. created a structured input features.
Model used: Logistic Regression (Scikit_learn). split data into training and validation set.
Evaluation: Validation accuracy 0.79, kaggle leaderboard score 0.75.
Result: model achieved  79% accuracy on validation data and 75% on kaggle test data.
